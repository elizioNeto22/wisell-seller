### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/elizioNeto22/wisell-seller.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Initialize the project
   ```sh
   npm start
   ```
